# Txchya AI

Txchya is a cyberpunk-futuristic AI assistant and OS for the Txchyon ecosystem.

## Setup

1. Clone this repo.
2. Create a `.env` file in `backend/` with your GPT-4o-mini API key:
   ```
   OPENAI_API_KEY=your_api_key_here
   ```
3. Install dependencies:
   ```bash
   cd backend
   npm install
   ```
4. Start the backend:
   ```bash
   node server.js
   ```
5. Embed `frontend/chat-widget.html` into txchyon.com.

## Usage

- Type messages into the chat widget to talk with Txchya.
- She references Txchyon blogs, Txchya music, and EverRank features.
- ElizaOS integration can be added later for advanced crypto tasks.
